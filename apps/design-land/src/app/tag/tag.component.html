Tag

Tags are compact visual indicators used to display short pieces of information, such as status, categories, or labels. They contain a text label, and optionally an icon and a dismiss button.

Overview

The tag component uses content projection to allow for flexible combinations of icons, labels, and interactive elements within a consistent container.

Types

daff-tag — Flexible tag container that can contain an icon, a label, and a dismiss button. The tag can be disabled by using the disabled property. Disabled tags cannot be dismissed.

Sizes

The size of a tag can be updated by using the size property. The size of tags will default to md if no size is defined. This can be updated to the three sizes tags support.

Supported Sizes: sm | md | lg

Theming

The default color of a tag is light gray, but it can be updated to one of the supported colors by using the color property.

Supported colors: primary | secondary | tertiary | dark | light | theme | theme-contrast

Status Indicators

Tags with status indicators can be used to distinguish the type of information they represent and their importance compared to other tags in the same context.

Supported statuses: warn | critical | success | info

Accessibility

The tag component uses semantic HTML elements and proper ARIA attributes to ensure accessibility. The delete button includes an aria-label="Delete" attribute and supports the disabled state with appropriate aria-disabled attributes.
